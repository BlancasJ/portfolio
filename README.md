# Portfolio

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Single-page layout with multiple sections.

## Tech Stack
- HTML5
- CSS3 (with responsive stylesheet)
- Vanilla JavaScript
- Font Awesome icons

## How to Run
Open `index.html` in a browser. No build step required.

## Features
- Responsive single-page design with hamburger navigation menu
- Sections: Home, About, Interests, Portfolio, Contact
- Neumorphic UI style with inner/outer shadow effects
- Social links (GitHub, email)
- Mobile-friendly with dedicated responsive CSS
